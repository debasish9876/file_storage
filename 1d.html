  • Write an algorithm to reverse a string.

    Write an algorithm to reverse a string in JavaScript.

        function reverseString(str) {
        var reversed = "";
        for (var i = str.length - 1; i >= 0; i--) {
            reversed += str[i];
        }
        return reversed;
    }
